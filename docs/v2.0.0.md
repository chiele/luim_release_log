# V2.0.0版本上线内容

2020.11.24

------

## 大部队ERP

### 新增

#### 1️⃣  新增运营-商品管理-拼多多系列店铺商品，拼多多商品受拼多多规则影响，不能立时禁用、限价等。

![1](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/1.png)

#### 2️⃣  新增销售单-销售渠道-拼多多系列店铺，包含筛选等所有相关功能。

![2](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/2.png)



#### 3️⃣  新增销售-本地库存定价-拼多多系列店铺，可批量改价和导出，规则无变化。

![3](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/3.png)

#### 4️⃣  新增销售-本地库存定价推送日志-拼多多系列店铺，定价修改后，会显示相关日志。

![4](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/4.png)

#### 5️⃣  新增供销-自动下单日志-拼多多系列店铺，包含日志中的所有操作和内容。

![5](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/5.png)

#### 6️⃣  新增供销-库存同步日志-拼多多系列店铺。

![6](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/6.png)

#### 7️⃣  新增供销-仓库维护-拼多多系列店铺，可进行店铺仓库维护。

![7](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/7.png)

#### 8️⃣  新增供销-同步参数-拼多多系列店铺（包含沙盒等），可参照之前操作流程进行拼多多店铺的修改。

![8](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/8.png)

#### 9️⃣  新增供销-自动下单参数-拼多多系列店铺，可参照之前操作流程进行拼多多店铺的修改。

![9](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/9.png)

#### 🔟  新增个性化配置-店铺配置，可选择对应店铺，选择后整个大部队相应的选择店铺的页面（销售单、售后单等），默认优先选中已配置的店铺，也可通过点击恢复默认按钮恢复全选。

![10](https://luim-public.oss-cn-zhangjiakou.aliyuncs.com/release_docs/v2.0.0/10.png)
